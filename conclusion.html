Conclusão

Após toda esta análise e caracterização com o apoio de várias ferramentas Bioinformáticas, conseguimos não só realizar a anotação funcional parcial do genoma da Legionella pneumophila str Philadelphia 1, considerando apenas os genes lpg2542-lpg2772, assim como obtivemos uma visão do poder da Bioinformática. Com isto foi-nos possível realizar a nossa própria avaliação a estes instrumentos enquanto se extraía informação destes. Foram criadas scripts para automatizar todo o processo de anotação funcional. Primeiro, fez-se uma análise de literatura para conhecermos melhor o organismo Legionella pneumophila. Analisando várias bases de dados construimos uma tabela detalhada acerca de todas as proteinas codificadas pelos genes em estudo. Depois, usou-se o BLAST para fazer as homologias dos genes, pois este pesquisa e compara dentro de bases de dados existentes sequências semelhantes às introduzidas, permitindo então verificar a existência de homologias com outros organismos. Recorrendo à ferramenta ClustalW e às sequencias homologas obtidas nos blasts construímos arvores filogenéticas. Estas servem principalmente para fazer uma análise para as proteínas de função desconhecida. Tendo em conta funções que se destacavam mais, foram usadas bases de dados mais específicas e criadas tabelas para cada função de interesse. Outra característica que também achamos relevante, foi perceber quais os genes com propriedades virulentas, nomeadamente a resistência a antibióticos, fatores de virulência e possíveis “drug targets”, para isso usou-se a base de dados PATRIC. Analisando outra base de dados, a NCBI CDD, pode-se obter mais informações sobre todas as proteínas, pois esta fornece-nos os domínios destas. ​Assim, ferramentas bioinformáticas como o “Biopython” aumentam a facilidade de procura de informação nas bases de dados apropriadas e obter resultados destas.
